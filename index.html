Documentation

Creating your all favorite game is always feels amazing. For assignment 3, I decided to take a step ahead from the chase game v1.0 that displayed my favorite red bird, the angry bird, to a more crazy one: Flappy Bird.

Who is Flappy Bird?

Flappy Bird is a bird that follows the law of gravity more than anything else on a computer screen. The bird tends to fall down unless the player keeps it flying. The bird loves to gather coins that are floating on the sky. However, its biggest fear are the columns shaped hurdles that comes in its way.

Design Features:

Unlike the Angry Bird in the chase game v1.0, Flappy Bird is created using advanced OOP approach. Everything that the player sees on the screen is an object that has its on attributes and methods that maintains its functionality.

The Game Class - Backbone

The game class is the backbone of this game. It keep tracks of all the instances and functionality of the game. It keeps it running when the Flappy Bird is doing well and stops it when Flappy dies. The game class is responsible for the motion on the screen. More on it later.

Use of Dynamic Arrays

Dynamic Arrays are always lovely because you don’t have to initiate them with a specific length. They grow as you grow. In this game, I am using two arrays: Platforms and Coins. These arrays store the randomly generated instances of Platforms or Hurdles and Coins. These arrays are later used by the Game Class to keep track of what is going on with each instance. This approach is also good as it saves hard coding or reading from a file, things that take time and computer scientists are lazy.

Functions

Guess what does flappy.jump(); do? I know that was easy as the function itself tells all about its functionality. In this game as everything is an object, it has its methods that easily do things that would take a lot of effort and lines of code. Yes, functions are used to make the code neat and reuse code.

HTML & CSS

Using HTML and CSS with p5.js, I have extended the functionality of the game, making it look professional. The buttons and the gravity slider are HTML elements, beautified by CSS, and operated by javaScript.

Physical Interface

Physical Interfaces are fun when it comes to games. In an interactive class, a group of us created a physical controller that uses a very different approach to manage key presses by the user. The interface uses different shapes drawn on pieces of paper with graphite pencil and Makey Makey. As the user touches one of the images on the paper, a circuit gets complete and the character moves in a particular direction. The user has to wear a conductive band to be part of the physical interface.
